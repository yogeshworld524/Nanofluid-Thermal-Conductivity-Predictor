# Nanofluid Thermal Conductivity Predictor

An engineering-focused machine learning project for predicting the thermal conductivity of nanofluids using both classical theoretical models and data-driven AI approaches.

## Features

* Maxwell model implementation
* Experimental data analysis
* Machine Learning prediction
* Streamlit web interface
* Thermal conductivity enhancement visualization

## Applications

* Heat Transfer Systems
* Refrigeration Engineering
* Thermal Management
* Energy Systems Research
* Nanotechnology Research

## Future Scope

* ANN-based prediction
* Deep Learning models
* Multi-objective optimization
* Nanorefrigerant property prediction
